# module03 — защита БД и целостность (SQLite)

Учебный проект: схема, триггеры, журнал аудита, логические роли, политика доступа.

## Быстрый старт

1. Установите [SQLite CLI](https://www.sqlite.org/download.html), добавьте `sqlite3` в `PATH`.
2. Из корня репозитория: `powershell -ExecutionPolicy Bypass -File scripts\init_sqlite.ps1`
3. Резервная копия: `scripts\backup_sqlite.ps1`
